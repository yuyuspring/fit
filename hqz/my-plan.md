## 训练注意事项（针对健康状况）

- **右脚踝**：深蹲/箭步蹲时注意重心不要偏左，右脚可略微朝前，避免踝关节过度内翻
- **髋关节**：训练前做髋关节活动度热身，避免突然扭转
- **腰突**：所有俯身动作（划船、硬拉）保持背部平直，不要弓背，重量从轻开始
- **颈椎**：俯身动作时保持颈部与脊柱一条直线，不要低头；起身时缓慢，避免头晕
- **低体位头疼**：俯身/下蹲后起身时放慢速度，配合呼气

## 渐进规则

### 新手进阶（优先级）
1. **掌握动作**：先用轻重量或徒手学动作标准
2. **增加次数**：每组稳定完成目标次数后，增加2次
3. **增加组数**：次数稳定后，从2组加到3组
4. **增加重量**：最后才加重（2.5kg）

### 示例
```
第1周：哑铃深蹲 2组×10次×5kg
第2周：哑铃深蹲 2组×12次×5kg
第3周：哑铃深蹲 3组×10次×5kg（加到3组，回到次数下限）
第4周：哑铃深蹲 3组×12次×5kg
```

## 营养建议（减脂期）

| 营养素 | 每日目标 |
|--------|----------|
| 热量 | 1400-1700 kcal |
| 蛋白质 | 99-136g（1.6-2.2g/kg） |
| 碳水 | 150-200g |
| 脂肪 | 40-55g |

### 减脂饮食要点
- 蛋白质每餐25-35g
- 多吃蔬菜增加饱腹感
- 减少精制碳水和糖
- 保持水分摄入2-3升/天
- 经期前后可适当增加碳水摄入
