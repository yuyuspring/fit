# 热身流程

> 训练前必做，5-10分钟

---

## 目的
- 提高体温和心率
- 激活目标肌群
- 增加关节活动度
- 预防受伤

---

## 通用热身（每次训练前）

| 动作 | 时长/次数 | 说明 |
|------|-----------|------|
| 开合跳 | 30秒 | 提高心率 |
| 高抬腿 | 30秒 | 激活腿部 |
| 手臂环绕 | 前10次+后10次 | 肩关节热身 |
| 腿摆动 | 每侧10次 | 髋关节热身 |
| 躯干旋转 | 每侧10次 | 脊柱活动度 |
| 猫牛式 | 10次 | 脊柱灵活性 |

---

---

## 热身流程示例

### 上肢训练日（5分钟）
```
1. 开合跳 30秒
2. 高抬腿 30秒
3. 手臂环绕 前后各10次
4. 肩胛骨前伸/后缩 10次
5. 俯卧撑 10次
6. 哑铃肩推（空手） 10次
→ 开始正式训练
```

### 下肢训练日（5分钟）
```
1. 开合跳 30秒
2. 高抬腿 30秒
3. 腿摆动 每侧10次
4. 徒手深蹲 15次
5. 臀桥 15次
6. 弓步走 每侧10次
→ 开始正式训练
```

## 冷却流程（训练后）

| 动作 | 时长 | 说明 |
|------|------|------|
| 慢走/原地踏步 | 2-3分钟 | 降低心率 |
| 股四头肌拉伸 | 每侧30秒 | 大腿前侧 |
| 腘绳肌拉伸 | 每侧30秒 | 大腿后侧 |
| 臀部拉伸 | 每侧30秒 | 臀部 |
| 胸部拉伸 | 30秒 | 胸肌 |
| 背部拉伸 | 30秒 | 背部 |
| 深呼吸 | 5次 | 放松身心 |

---

## 注意事项

1. 热身不是拉伸，动态为主
2. 热身强度逐渐递增
3. 感觉身体热起来即可
4. 天冷时可延长热身时间
5. 训练后做静态拉伸

# 休息日活动指南

> 完全休息不等于躺平，适当活动促进恢复

---

## 休息日类型

### 主动恢复日
适合：训练后第二天，感觉轻微疲劳

| 活动 | 时长 | 说明 |
|------|------|------|
| 散步 | 20-30分钟 | 促进血液循环 |
| 泡沫轴放松 | 10-15分钟 | 放松紧张肌肉 |
| 轻度拉伸 | 10分钟 | 改善柔韧性 |
| 瑜伽 | 20-30分钟 | 身心放松 |

### 完全休息日
适合：感觉明显疲劳，或连续训练后

| 活动 | 时长 | 说明 |
|------|------|------|
| 轻度活动 | 10-15分钟 | 避免久坐 |
| 深呼吸/冥想 | 5-10分钟 | 降低压力 |
| 充足睡眠 | 7-9小时 | 身体修复 |

## 泡沫轴放松流程

### 上肢放松（5分钟）
| 部位 | 时长 | 动作 |
|------|------|------|
| 背阔肌 | 每侧30秒 | 侧卧滚动 |
| 胸椎 | 30秒 | 仰卧滚动 |
| 三角肌 | 每侧30秒 | 侧卧滚动 |

### 下肢放松（10分钟）
| 部位 | 时长 | 动作 |
|------|------|------|
| 股四头肌 | 每侧30秒 | 俯卧滚动 |
| 腘绳肌 | 每侧30秒 | 坐姿滚动 |
| 臀肌 | 每侧30秒 | 坐姿滚动 |
| 小腿 | 每侧30秒 | 坐姿滚动 |
| IT带 | 每侧30秒 | 侧卧滚动 |

### 放松技巧
1. 找到痛点，停留20-30秒
2. 缓慢滚动，不要快速来回
3. 配合深呼吸，放松身体
4. 力度适中，疼痛感<7分

|------|------|------|

| 背部拉伸 | 30秒 | 猫牛式 |
| 肩部拉伸 | 每侧30秒 | 交叉手臂 |
| 三头肌拉伸 | 每侧30秒 | 头顶拉伸 |

### 下肢拉伸
| 动作 | 时长 | 说明 |
|------|------|------|
| 股四头肌拉伸 | 每侧30秒 | 站姿拉伸 |
| 腘绳肌拉伸 | 每侧30秒 | 坐姿前屈 |
| 臀部拉伸 | 每侧30秒 | 仰卧4字拉伸 |
| 小腿拉伸 | 每侧30秒 | 弓步拉伸 |

### 拉伸要点
1. 拉伸到有感觉但不疼痛
2. 保持正常呼吸，不要憋气
3. 每个动作保持30秒
4. 不要弹震式拉伸

---

## 休息日营养

### 重点
- 蛋白质摄入不变（帮助肌肉修复）
- 碳水可适当减少（消耗降低）
- 保持水分摄入

### 建议
- 蛋白质：1.6-2.0g/kg体重
- 热量：比训练日低200-300kcal
- 水分：2-3升/天

---

## 休息日禁忌

❌ 久坐不动（超过2小时不活动）
❌ 完全躺平一整天
❌ 熬夜（影响恢复）
❌ 饮酒（影响蛋白质合成）
❌ 过度训练（感觉疲劳还坚持）

---

## 休息日信号

### 需要主动恢复的信号
- 轻微肌肉酸痛
- 训练后感觉还好
- 睡眠质量正常

### 需要完全休息的信号
- 持续疲劳感
- 睡眠质量差
- 训练动力低下
- 静息心率升高
- 关节不适
