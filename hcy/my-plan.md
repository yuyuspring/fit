## 渐进规则

### 每周进步方式（按优先级）
1. **增加次数**：在当前重量下做到组数上限 → 下周增加2次
2. **增加组数**：次数稳定后 → 增加1组
3. **增加重量**：组数稳定后 → 增加2.5kg

### 示例进阶
```
第1周：哑铃卧推 4组×8次×10kg
第2周：哑铃卧推 4组×10次×10kg
第3周：哑铃卧推 4组×12次×10kg
第4周：哑铃卧推 4组×10次×12.5kg（回到次数下限，加重）
```

## 休息日安排

### 🟢 非训练日康复（方案B·约15分钟）— 全套康复，含离心+泡沫轴

> 休息日精力充足，做完整版。每部位选1-2个动作即可。

**跟腱**：离心下落 3×15 + 站姿拉伸 30秒/侧
**ITB**：交叉腿拉伸 30秒/侧 + 泡沫轴股四外侧+臀中肌 各1分钟
**腰椎**：鸟狗式 3×10/侧 + 仰卧抱膝拉伸 30秒/侧
**腹股沟**：髂腰肌拉伸 30秒/侧 + 蛤蜊式 2×12/侧
**胸椎**：开书式（重点右侧）+ 穿针式 + 猫牛式 各2组
**肩颈+右肩**：颈部拉伸 + 收下巴 + 肩胛环绕 + 弹力带外旋 各1组

### 主动恢复（可选）
- 散步20-30分钟
- 轻度拉伸

### 完全休息
- 保证7-9小时睡眠
- 充足蛋白质摄入
