# 长期健身计划框架

> 支持多人独立训练计划和记录

---

## 使用流程

```
1. 填写 baseline.md → 基线评估（一次性）
2. 查看 shared/exercises.md → 学习动作要点
3. 按 week-*.md → 执行本周训练
4. 记录 weekly-checkin.md → 打卡记录
5. 参考 shared/adjustment-rules.md → 调整下周计划
6. 更新 progress-log.md → 追踪长期进度
```

## 渐进规则

### hcy（增肌+力量）
按优先级递进：
1. 增加次数：当前重量下做到组数上限 → 下周增加2次
2. 增加组数：次数稳定后 → 增加1组
3. 增加重量：组数稳定后 → 增加2.5kg

### hqz（减脂+体态）
按优先级递进：
1. 掌握动作：先用轻重量学动作标准
2. 增加次数：做到15次后再考虑加重
3. 增加组数：次数稳定后增加1组
4. 增加重量：最后才加重（2.5kg）

## 重要提示

1. **训练前必做热身** → 见 shared/warmup.md
2. **动作不标准时减重** → 参考 shared/exercises.md
3. **每周记录打卡** → 填写 weekly-checkin.md
4. **感觉疲劳时减量** → 参考 shared/adjustment-rules.md
